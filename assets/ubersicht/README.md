# Übersicht Widget (LeetCode Streak)

## Install Übersicht
Download from the official site:
https://tracesof.net/uebersicht/

## Install the Widget
1. Open Finder
2. Go to: `~/Library/Application Support/Übersicht/widgets`
3. Copy the folder `leetcode-streak.widget` from this repo:
   `assets/ubersicht/leetcode-streak.widget`

## Configure
Open `index.jsx` and set:
```
const USERNAME = "Uvii555";
```

Your streak widget will appear on the desktop. It refreshes every hour.
